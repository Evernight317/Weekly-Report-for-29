## 二、核心学习内容

### 2.1 Java 入门（黑马 Java 0-80 节）

有 C++ 基础快速过一遍基础语法，重点攻克面向对象、核心类库、集合三大模块，反射、注解等进阶内容延后学习。熟练使用 IDEA 完成项目创建、断点调试，利用集合接收数据库批量查询数据，通过类封装数据表实体信息。

### 2.2 MySQL 数据库（黑马 MySQL 0-26 节）

掌握建库、建表、增删改查基础 SQL 语句，使用 Navicat 可视化工具管理数据库。

简化建库建表语句：

```sql
CREATE DATABASE java_demo;
USE java_demo;
CREATE TABLE user(id INT PRIMARY KEY AUTO_INCREMENT,name VARCHAR(20));
```

### 2.3 Java 连接 MySQL 实现 CRUD（本周核心）

1. Maven 驱动依赖配置

```xml
<dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java</artifactId>
    <version>8.0.46</version>
</dependency>
```

1. 数据库工具类关键代码片段

```java
String url="jdbc:mysql://localhost:3306/java_demo?serverTimezone=Asia/Shanghai";
static {Class.forName("com.mysql.cj.jdbc.Driver");}
```

1. CRUD 核心逻辑

   ```
   PreparedStatement
   ```

   占位符执行 SQL，分别封装新增、删除、修改、查询方法，新建测试类调用功能，所有数据操作可同步持久化到 MySQL 数据表。

## 四、问题与解决方案

1. 数据库连接抛出时区报错

   错误配置：`jdbc:mysql://localhost:3306/java_demo`

   解决：url 末尾追加`serverTimezone=Asia/Shanghai`参数。

2. 直接拼接字符串构造 SQL 语句

   错误写法：`"insert user(name) values('"+name+"')"`

   解决：使用`?`占位符传递参数，规避语法错误与 SQL 注入风险。

3. 未释放数据库连接资源

   现象：多次运行程序后连接失效、运行卡顿

   解决：工具类统一封装资源关闭方法，操作完成后释放连接、结果集。
